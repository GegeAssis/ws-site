# W.S. Assessoria – Site institucional

Site simples de uma página para apresentação da W.S. Assessoria.

## Como usar

- Coloque `index.html` e `logo-ws.png` na mesma pasta.
- Faça upload dessa pasta em qualquer hospedagem estática (Vercel, Netlify, etc).
- Edite o `index.html` para preencher:
  - Contato
  - E-mail
  - CNPJ
